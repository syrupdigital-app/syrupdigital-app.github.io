01

Pendahuluan

Selamat datang di MapCatalog, aplikasi katalog properti berbasis peta yang dikembangkan oleh Syrup Digital. Aplikasi ini dirancang untuk membantu agen penjualan properti di Indonesia menampilkan dan berbagi informasi produk kepada calon pembeli secara efisien.

Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi informasi Anda ketika menggunakan aplikasi MapCatalog di perangkat Android Anda.

Dengan menggunakan MapCatalog, Anda menyetujui praktik yang dijelaskan dalam Kebijakan Privasi ini. Mohon baca dokumen ini dengan seksama sebelum menggunakan aplikasi.
02

Data yang Dikumpulkan

MapCatalog mengumpulkan beberapa kategori data untuk memberikan layanan yang optimal:

  • Data Lokasi: Koordinat GPS perangkat Anda (latitude & longitude) untuk menampilkan posisi Anda di peta dan menemukan properti terdekat. Data lokasi hanya diakses saat aplikasi aktif digunakan (foreground only).
  • Data Properti: Informasi produk yang Anda tambahkan sendiri, termasuk nama, harga, deskripsi, status, foto, dan koordinat lokasi properti.
  • Foto dan Media: Foto properti yang Anda unggah dari galeri perangkat, disimpan secara lokal di penyimpanan internal aplikasi.
  • Data Profil Pengguna: Nama agen, nomor WhatsApp, dan nama perusahaan yang Anda isi secara manual di menu Profil.
  • Data Teknis: Informasi perangkat (versi Android, model) yang dikumpulkan secara anonim oleh layanan iklan pihak ketiga (AdMob).
03

Tujuan Penggunaan Data

Data yang kami kumpulkan digunakan semata-mata untuk keperluan berikut:

  • Menampilkan posisi Anda dan properti di atas peta interaktif
  • Menyimpan dan mengelola katalog properti Anda secara lokal
  • Menghasilkan brosur PDF dan pesan WhatsApp berisi informasi properti
  • Menampilkan iklan yang relevan melalui Google AdMob
  • Mengirim pembaruan aplikasi melalui mekanisme Google Play In-App Updates

Kami tidak menggunakan data Anda untuk kepentingan pemasaran pihak ketiga, profiling komersial, atau tujuan lain di luar fungsi inti aplikasi.

04

Penyimpanan Data

Seluruh data katalog properti, foto, dan profil pengguna disimpan secara lokal di perangkat Anda menggunakan Capacitor Filesystem API. Data ini tidak dikirimkan ke server eksternal milik Syrup Digital.

Lokasi penyimpanan:

  • Database produk: file JSON di direktori privat aplikasi (DATA directory)
  • Foto properti: direktori mapcatalog_photos/ di penyimpanan eksternal aplikasi
  • Preferensi pengguna: localStorage di WebView Capacitor
Karena data disimpan lokal, menghapus atau menguninstall aplikasi akan menghapus semua data katalog Anda secara permanen. Lakukan backup secara berkala jika diperlukan.
05

Berbagi Data dengan Pihak Ketiga

MapCatalog tidak menjual, menyewakan, atau memperdagangkan data pribadi Anda kepada pihak mana pun. Berbagi data terjadi hanya dalam skenario berikut:

  • Berbagi oleh pengguna sendiri: Saat Anda menekan tombol "Bagikan via WhatsApp" atau "Ekspor PDF", data properti dibagikan melalui aplikasi yang Anda pilih. Proses ini sepenuhnya diinisiasi oleh Anda.
  • Google AdMob: SDK iklan Google mungkin mengumpulkan identifikasi perangkat dan data penggunaan secara anonim untuk keperluan penayangan iklan. Lihat Kebijakan Privasi Google untuk detail lebih lanjut.
  • Google Play Services: Digunakan untuk mekanisme pembaruan aplikasi otomatis (In-App Updates). Tidak ada data pribadi yang dikirimkan.
06

Izin Perangkat yang Diminta

MapCatalog meminta izin berikut saat dijalankan:

  • Lokasi (Fine Location): Diperlukan untuk menampilkan posisi GPS Anda di peta. Tanpa izin ini, fitur "Lokasi Saya" tidak akan berfungsi.
  • Kamera / Baca Penyimpanan: Digunakan untuk memilih foto properti dari galeri. Aplikasi tidak mengakses galeri tanpa tindakan eksplisit dari Anda.
  • Tulis ke Penyimpanan: Diperlukan untuk menyimpan file PDF yang diekspor ke perangkat Anda (Android 9 ke bawah).
  • Internet: Dibutuhkan untuk memuat tile peta (OpenStreetMap, Esri), menampilkan iklan AdMob, dan pembaruan aplikasi.

Semua izin dapat Anda kelola kapan saja melalui menu Pengaturan > Aplikasi di perangkat Android Anda.

07

Layanan Pihak Ketiga

MapCatalog mengintegrasikan layanan pihak ketiga berikut yang memiliki kebijakan privasi masing-masing:

  • OpenStreetMap & Nominatim: Penyedia data peta dan layanan geocoding (pencarian alamat). Data pencarian lokasi Anda dapat dikirim ke server Nominatim untuk memproses hasil. Lihat Kebijakan Privasi OSM.
  • Esri (ArcGIS): Penyedia tile peta satelit. Permintaan tile dikirmkan ke server Esri. Lihat Kebijakan Privasi Esri.
  • Google AdMob: Platform iklan mobile Google. Dapat mengumpulkan data perangkat anonim. Lihat Kebijakan Privasi Google.
  • Google Play Core (In-App Updates): Digunakan untuk pembaruan aplikasi in-app tanpa menutup aplikasi.
08

Keamanan Data

Kami mengambil langkah-langkah yang wajar untuk melindungi data Anda, termasuk:

  • Data katalog disimpan di direktori privat aplikasi yang tidak dapat diakses aplikasi lain tanpa root.
  • Komunikasi jaringan menggunakan HTTPS/TLS untuk semua permintaan eksternal.
  • Aplikasi tidak menyimpan kredensial pengguna (password, token autentikasi) karena tidak memerlukan akun.

Namun, tidak ada metode transmisi atau penyimpanan elektronik yang 100% aman. Kami tidak dapat menjamin keamanan absolut atas data yang disimpan di perangkat Anda apabila perangkat tersebut hilang, dicuri, atau diakses pihak tidak berwenang.

09

Hak Pengguna

Karena seluruh data disimpan secara lokal di perangkat Anda, Anda memiliki kendali penuh atas data tersebut:

  • Akses: Anda dapat melihat semua data katalog langsung melalui antarmuka aplikasi.
  • Koreksi: Anda dapat mengedit atau memperbarui informasi produk dan profil kapan saja.
  • Penghapusan: Anda dapat menghapus produk satu per satu, atau menghapus semua data dengan menguninstall aplikasi.
  • Portabilitas: Data tersimpan dalam format JSON yang dapat diekspor (fitur akan tersedia di pembaruan mendatang).

Untuk pertanyaan terkait data yang diproses oleh layanan pihak ketiga (AdMob, dll.), silakan hubungi penyedia layanan tersebut secara langsung.
